# Music_player_mini
Music_player with available function: 
  1. Render songs
    2. Scroll top
    3. Play / pause / seek
    4. CD rotate
    5. Next / prev
    6. Random
    7. Next / repeat when ended
    8. Active song
    9. Scroll active song into view
    10. Play song when click
    11. Volume button
    12. Muted button
    13. PlaybackRate button
    14. Thumbnail background match with currentsong - done but not blur
